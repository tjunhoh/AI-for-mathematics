Large language models (LLMs)

Transformer models (the hot new thing!) are mainly used to train LLMs.

General purpose LLMs:
  • ChatGPT - OpenAI's LLM
  • Microsoft Copilot - AI companion for Microsoft 365 apps
  • Llama 2 - open source LLM
Customized LLMs:
  • GPTs - storefront for customized ChatGPTs
  • Wolfram GPT - ChatGPT customized for mathematics and data visualization
  • GitHub Copilot - AI-based code suggestions
  • NovelAI - storytelling
  • Friends & Fables - AI game master for tabletop RPGs
  • AI Dungeon - infinitely generated text adventure
LLM (and other generative AI) communities
  • Hugging Face

Diffusion models

Diffusion models can be used for text-to-video and text-to-image generation.

Text-to-video:
  • Lumiere
Text-to-image:
  • Stable Diffusion
  • DALL·E - also available as a custom version ChatGPT
  • Midjourney
